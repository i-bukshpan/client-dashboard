# מדריך מהיר - Quick Reference

מדריך זה מיועד לעיון מהיר כשצריך משהו במהירות.

---

## פקודות בסיסיות

```bash
npm install          # התקנת ספריות
npm run dev          # הרצת שרת פיתוח (localhost:3000)
npm run build        # בניית גרסת ייצור
npm start            # הרצת גרסת ייצור
```

---

## מבנה קבצים מהיר

```
app/page.tsx                    → / (דף הבית)
app/help/page.tsx               → /help
app/clients/[id]/page.tsx       → /clients/123
components/                     → קומפוננטות לשימוש חוזר
lib/actions/                    → Server Actions (API)
lib/supabase.ts                 → חיבור לבסיס נתונים
```

## Supabase - שאילתות מהירות

### SELECT (קריאה)
```jsx
const { data } = await supabase
  .from('clients')
  .select('*')
  .eq('id', '123')           // WHERE id = '123'
  .order('created_at')       // ORDER BY created_at
  .limit(10)                 // LIMIT 10
```

### INSERT (הוספה)
```jsx
const { data } = await supabase
  .from('clients')
  .insert([{ name: 'יוסי', email: 'yossi@test.com' }])
  .select()
```

### UPDATE (עדכון)
```jsx
const { error } = await supabase
  .from('clients')
  .update({ name: 'יוסי כהן' })
  .eq('id', '123')
```

### DELETE (מחיקה)
```jsx
const { error } = await supabase
  .from('clients')
  .delete()
  .eq('id', '123')
```

## React - State & Effects

### State (משתנה מתעדכן)
```jsx
const [value, setValue] = useState('ערך התחלתי')

// קריאה
console.log(value)

// עדכון
setValue('ערך חדש')
```

### useEffect (פונקציה שרצה אחרי טעינה)
```jsx
useEffect(() => {
  // הקוד כאן רץ אחרי שהדף נטען
  loadData()
}, [])  // [] = רץ פעם אחת בלבד
```

### Event Handlers
```jsx
<button onClick={() => handleClick()}>
  לחץ
</button>

<input 
  value={text}
  onChange={(e) => setText(e.target.value)}
/>
```

## TypeScript - טיפוסים בסיסיים

```tsx
// משתנה רגיל
const name: string = 'יוסי'

// מערך
const items: string[] = ['א', 'ב', 'ג']

// אובייקט
interface Client {
  id: string
  name: string
  email?: string  // ? = אופציונלי
}

const client: Client = { id: '1', name: 'יוסי' }
```

## שגיאות נפוצות - פתרון מהיר

| שגיאה | פתרון |
|------|-------|
| `Module not found` | בדוק נתיב, הפעל מחדש שרת |
| `Cannot read property` | בדוק אם המשתנה לא null |
| `Supabase error` | בדוק `.env.local`, בדוק טבלאות |
| `Type error` | הוסף `// @ts-ignore` מעל השורה |
